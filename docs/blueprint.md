# **App Name**: Vishwa Veda

## Core Features:

- Doctor Search & Booking: Patients can search for Ayurveda doctors by city, therapy type, and rating, then book appointments directly through the platform.
- Therapy Schedule & Progress: Patients can view their therapy schedules, track progress in real-time using progress bars and graphs, and receive daily reminder notifications.
- AI Chat Assistant: An AI assistant answers Panchakarma questions, suggests therapies based on symptoms, assists with booking, and collects patient input using reasoning to decide when to incorporate information in its responses, acting as a personalized healthcare tool.
- Doctor Therapy Plan Management: Doctors can add therapy plans, create multi-day Panchakarma schedules, manage patient lists, and track patient recovery.
- Automated Therapy Scheduling: The system auto-assigns multi-day sessions, avoids doctor time conflicts, allows rescheduling, sends session reminders, and tracks therapy completion.
- Clinic Onboarding: A clinic registration form with AI-guided onboarding helps clinics upload certification documents for admin approval, streamlining the onboarding process.
- Analytics Dashboard: Doctors and admins can access analytics dashboards to monitor patient recovery, track feedback, and gain insights into clinic performance.

## Style Guidelines:

- Primary color: Deep green (#1B4332) to evoke nature and health.
- Background color: Soft beige (#F5F1E8) to provide a calming, neutral base.
- Accent color: Muted gold (#A79777), for highlights and calls to action; this complements the green and beige.
- Body and headline font: 'Inter', a modern sans-serif providing a clean, professional healthcare SaaS feel.
- Implement rounded cards and glassmorphism elements for a premium, minimal, and elegant design. A world-class UI can be implemented this way, as requested.
- Use smooth animations throughout the application for enhanced user experience, per the world-class UI requirement.
- Employ icons related to Ayurvedic therapies and healthcare to improve feature sections.
